## 📂 Organização do Repositório

O projeto está dividido entre a base teórica em LaTeX e as implementações práticas em Java:

*   **`/latex`**: Documentação técnica contendo os enunciados, questões reflexivas e requisitos pedagógicos (`main.tex`).
*   **`/src`**: Código-fonte Java organizado por módulos temáticos.
    *   `01_recursividade`: Pilha de chamadas e recursão.
    *   `02_array_primitivo`: Alocação contígua e shifts.
    *   `03_arraylist`: Listas dinâmicas e Collections API.
    *   `04_generics`: Type Safety e Bounded Types.
    *   `05_06_07_08_listas`: Implementações de listas encadeadas (Simples, Duplas, Circulares).

---

## 🧠 Metodologia de Estudo

Cada tópico da lista segue um escalonamento de dificuldade:
1.  **Conceito (Warm-up):** Fundamentação básica.
2.  **Prática (Core):** Aplicação em cenários reais.
3.  **Desafio (Expert):** Problemas complexos e tratamento de erros.

> [!TIP]
> **Dica de Monitoria:** Não tente apenas copiar o código. Use o nosso **Dashboard** para entender os "Gotchas" e erros comuns de cada estrutura antes de codificar.
